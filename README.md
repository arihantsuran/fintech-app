# fintech-app
A modern full-stack financial management platform built to explore real-world software engineering concepts including secure authentication, financial transactions, budgeting, analytics and cloud deployment. 

## Tech Stack
Frontend
-React 
-TypeScript
-Tailwind CSS

Backend
-Express.js
-Node.js

Database
-PostgreSQL

DevOps
-Docker
-GitHub Actions

## Planned Features
-User Auth
-Multiple Accounts
-Income & Expense Tracking
-Budget Management
-Financial Analytics Dashboard
-Transaction History
-Search & Filters
-Rest API
-Responsive UI

## Goals
This project is being developed to strengthen my backend engineering, database design, system architecture, and full-stack development skills while applying best practices used in production software.

The development journey will be documented through GitHub commits and LinkedIn updates.
